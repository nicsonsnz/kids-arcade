# SPEC — 蛇蛇星球 Noodle Stars（目录 `snake-io/`）

大蛇吃豆对战。吃发光豆变长变粗，用身体逼别的蛇撞上来；撞到别人身体即死，
死蛇化作一地光豆。无尽模式，目标登顶排行榜。深空糖果主题，主色 `#5b3df5`（星空紫）。短名「蛇蛇星球」。

## 1. 世界
- 圆形竞技场半径 2300 世界单位(≈px)，中心 (0,0)。背景深紫夜空 `#151030`：预绘离屏平铺层 = 微弱六边形网格 + 两层视差星点（远层 0.5×、近层 0.8× 镜头速度）。
- 边界：半透明发光红色环带（预烘焙渐变条带，禁 shadowBlur），头部越界即死。

## 2. 蛇的模型（研究报告 B.1 实现）
- 头部每逻辑步前进；轨迹点每 2u 一个（插值保持均匀），trim 到所需长度；身体段沿轨迹每 `seg=10u` 弧长取样。
- `mass` 标量：段数 `segs = min(18 + floor(mass/2.2), 420)`；半径 `r = 11 + 17*(1 - 1/(1 + mass/600))`（11→28）。
- 速度：基础 168 u/s，加速 336 u/s；转向率 `3.5 * (0.35 + 0.65*(11/r)^1.2) rad/s`（越粗转弯越钝），加速时再 ×0.65（高速大弧）。
- **加速**：按住即加速；条件 mass>18；消耗 14 mass/s，每 0.22s 从尾部掉一颗 value=2 的小豆（约回收 30%）；mass 触底自动断开。
- 自身碰撞无害（可以自由盘绕自己）。

## 3. 食物
对象池 + 空间哈希（哈希格 176u）。恒定豆量 ~420 颗随机散布（镜头外也存在；密度中心略高）。
- 小豆 value 1–3：半径 4–6，柔和糖果色发光点（预烘焙光晕 sprite，`lighter` 叠加），轻微脉动（正弦缩放，相位随机）。
- 大光珠 value 15：半径 10，占比 ~4%，双层光晕。
- 星蝶 value 40：全场同时 3–5 只，缓慢漂移，蛇头靠近 140u 内会朝反方向加速逃逸（速度 200），通常需要加速才能追上；小翅膀两枚三角形拍动。
- 死亡遗珠：死蛇沿身体每 3 段掉一颗 value = clamp(victimMass*0.55/n, 4, 30) 的大珠（上限 80 颗），10s 后开始闪烁、15s 消失（防地图堆积）。
- 磁吸：豆与蛇头距离 < r*3.2 时被吸向头（速度 240 递增），入口 pop 音（连续吃 combo 升调，0.8s 无进食重置）。

## 4. 碰撞与死亡
- 空间哈希（176u 格）存所有蛇的身体段；蛇头 vs 外蛇段圆检测，容差 -4u（略宽容）。头对头：双亡。出界：死。
- 死亡：身体化为遗珠 + 爆裂粒子 + 屏震（幅度随体型）+ 噪声爆破音；击杀者 toast「你吃掉了 XX！」与 +kill。
- 玩家死亡结算：本局长度、击杀、最高名次、最高纪录对比、鼓励文案、大「再来一局」（无复活机制，重开要快：≤2 次点击 3 秒内回到场上）。

## 5. Bot AI（bots.js）
- 恒定 13 条 AI 蛇；大小分布偏小（spawn mass 8–120，10% 概率 300–700 的大蛇）；死后 2–4s 换名重生保持人口。20Hz 决策 + 0.1–0.35s 反应延迟快照 + 性格向量 `aggression/caution/greed/boostiness/skill`。
- FSM 优先级：AVOID(墙/身体, 前向锥采样空间哈希，选最大空隙方向) > FLEE(更大蛇头近距逼近，逃向开阔处，boostiness 高则加速) > HUNT(自己 ≥1.3× 目标且距离近：预测目标头部前方位置切入拦截；skill 低会领先量失准) > COIL(目标 ≤0.4× 自己且很近：绕圈收紧，维持 6–10s 后放弃) > SEEK_FOOD(greed 半径内豆簇质心，死亡遗珠权重×3，星蝶只有 boostiness 高的追) > WANDER(随机巡航点 + heading 噪声)。
- 迟滞：状态切换 ≥0.5s 间隔；低概率漏看威胁（1-skill）制造可乘之机。
- 名字池 ≥24 个原创可爱名（面条侠、彩虹糖、流星锤、软糖将军…），皮肤从池里轮换。

## 6. 渲染（render.js）
层序：背景平铺 → 边界环 → 豆(光晕 sprite 批量) → 蛇 → 头/眼 → 粒子飘字 → HUD。
- **蛇身**：沿段点把轨迹 polyline 一次 stroke（lineWidth=2r, round cap/join）画底色，再叠一条 0.55r 宽、颜色亮 30%、y-2 偏移的高光 stroke；花纹皮肤 = 每 4 段在段点上叠一枚深/浅交替圆斑（同色系）。加速时：整蛇外圈叠 `lighter` 光晕 stroke（预烘焙线性渐变色），并从尾部冒火花粒子。
- 皮肤 6 款：橙黄环纹、蓝白冰纹、粉紫渐变、青柠条纹、彩虹循环（每段 hue+12°）、金色龙纹（解锁条件见 §8）。颜色沿段索引取模映射。
- **头**：比身体半径大 8% 的圆 + 两只朝航向前视的大眼（白珠+黑瞳+高光点，转向时瞳孔偏向目标方向）；张嘴细节：靠近食物 60u 内画张开的小嘴弧。名牌白字浮于头上 22u。#1 戴金冠。
- 镜头外段剔除（段点粗判 AABB）。
- 小地图：右下 120px 圆形：暗底 + 所有蛇按体型画 2–4px 色点 + 玩家白点脉冲 + 边界圈。3Hz 重绘。

## 7. 相机
- 跟头 exp 平滑 k=9；缩放 `zoom = clamp(1.05 * (14/r)^0.55, 0.5, 1.1)` exp 平滑 k=2.5（成长时缓缓拉远，权力感）。击杀/死亡震屏。

## 8. 操控 / HUD / 界面
- 触控：**动态摇杆**（任意落指出现摇杆，同圈地）控制方向；右下 84px 圆形「⚡加速」按钮（按住生效，safe-area 之上）；**第二根手指按住屏幕任意处也=加速**（两种都支持）。桌面：鼠标位置=目标方向（相对头部），按住左键/空格=加速。
- HUD：左上「长度 1234」大数字（弹跳更新）+ 小字最高纪录；右上排行榜前 10（名+长度，玩家行高亮，#1 👑，DOM 2Hz）；右下小地图+加速按钮；左下击杀 toast 堆栈；右上角暂停。
- 开始界面：LOGO「蛇蛇星球」+ 皮肤横排选择（锁定的显示条件）+ 最高纪录 + 大开始键 + 静音。
- 皮肤解锁（best length）：橙黄默认；1000 冰纹；2500 粉紫；5000 青柠；8000 彩虹；15000 金龙。
- 里程碑：本局长度首破 500/1000/2500/5000… 时全屏轻量 confetti + 上行琶音 + 「新纪录！」/「突破 xxxx！」飘字。

## 9. 音效
吃豆 pop（combo 升调）、大珠叮咚、星蝶捕获双音、加速持续气流声（低通噪声 loop，按住时淡入）、击杀爆破+短琶音、死亡下滑、里程碑琶音、按钮 pop。BGM：太空感 3 声部慢琶音循环（正弦+三角），音量 0.1。
