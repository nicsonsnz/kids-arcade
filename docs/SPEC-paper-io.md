# SPEC — 圈地大冒险 Popsicle Land（目录 `paper-io/`）

圈地吞并类游戏。玩家操控一个「冰棒小车」在纸面世界圈地：离开自己领地会拖出轨迹，
绕一圈回到领地即把包住的区域全部占为己有；轨迹被碰即死。目标：占领 100% 地图。
主色 `#ff8a3d`（暖橙）。短名「圈地大冒险」。

## 1. 世界与数据模型
- 方形地图 3200×3200 世界单位；格子 8u → 网格 400×400 = 160,000 cells，`Uint8Array grid`（0=中立，1..N=owner id）。
- 并行数组：`trailOwner Uint8Array`、`trailStep Uint16Array`（本次出行的步序号，自碰检测用）。
- `counts Int32Array` 每 owner 领地格数，唯一修改点 `setOwner(cell,newOwner)` 内增减；`percent = counts[id]/160000*100`。
- 出生领地：半径 18 格的圆（≈1017 格 ≈ 0.64%，对齐原版 0.66% 开局）。玩家出生在地图中心附近，bot 出生在中立区随机点（与任何领地、玩家保持 ≥40 格距离）。

## 2. 移动与操控
- 恒速自动前进 150 u/s（玩家与 bot 同速），只控方向；转向速率上限 3.8 rad/s（平滑追踪目标角，不允许瞬间掉头）。
- 触控：**任意位置落指即虚拟摇杆**——落点为杆心，拖动方向 = 目标航向，死区 10px，拖离 56px 达最大灵敏；抬指保持当前航向。屏幕上画出半透明摇杆底盘+杆头跟随手指（好看且可读）。
- 桌面：WASD/方向键（8 向），或按住鼠标拖动（指针相对角色方向 = 目标航向）。
- 地图边界：**滑墙不死**——把头部位置钳制在边界内并沿墙滑动；边界可作为封口墙（贴边圈地只需封一侧）。

## 3. 轨迹与占领（核心算法，照研究结论实现）
- 头部每逻辑步移动后，用 supercover DDA 把经过的所有格子盖章到 `trailOwner/trailStep`（防隧穿）；同时记录轨迹点列表（每 ~8u 一点）用于渲染。
- 在自己领地内：不产生轨迹、无敌。踏出领地那一刻开始记轨迹。
- **占领触发**：头部所在格从非自己领地变为自己领地（回家）时：
  1. bbox = (自己领地 ∪ 本次轨迹) 外扩 1 格；
  2. 从 bbox 边框把「非本人领地且非本人轨迹」的格子做 4 连通洪水填充标记 reachable；
  3. bbox 内所有 open 且 !reachable 的格子 `setOwner(己)`——中立与敌方领地一并吞并；
  4. 轨迹格全部 `setOwner(己)`，清空轨迹。
  5. 若某受害者 `counts==0`：该 bot 直接死亡（爆炸+移除）。
- 占领后计算本次增量 Δ%，飘字 `+X.X%`（保留 1 位小数）并播放占领音效；Δ% ≥3 时额外 confetti + 轻微屏震。

## 4. 生死规则（严格按此表）
| 情形 | 结果 |
|---|---|
| 任何头部进入他人轨迹格 | 轨迹主人死，碾过者得 1 击杀 |
| 头部进入**自己**轨迹格（trailStep 差 > 4 的旧格） | 自己死 |
| 两头相撞（圆距 < 头半径和），双方都在领地外 | 双方死 |
| 两头相撞，一方在自己领地内 | 领地内一方活，另一方死 |
| 在自己领地内 | 无敌 |
- 死亡时：轨迹清除；领地在 0.8s 内淡出为中立；角色 3 段式死亡演出（轨迹消失→角色爆裂粒子→领地淡出）＋噪声爆破音；镜头小震。
- **玩家死亡**：弹出复活面板——每局一次免费「复活」（5 秒倒计时按钮，保留死亡前领地与百分比，在领地质心重生并有 2s 无敌闪烁）；用过或超时则结算界面。bot 死亡：3–5s 后以新身份（换名换色）在中立区重生。

## 5. Bot AI（bots.js）
- 恒定人口 7 个存活 bot；死后延时重生，永不出生在玩家领地。20Hz 决策。
- 每个 bot 出生时抽性格：`aggression/caution/skill ∈ [0,1]`（正态偏中）；作用于出圈深度、逃跑阈值、反应延迟（0.1–0.4s 的延迟世界快照）、瞄准噪声。
- FSM 优先级：FLEE > DEFEND > RETURN > RAID > EXPAND：
  - EXPAND：规划一个出圈弧/矩形路径（深度 = f(1-caution)，8–40 格），沿路径点行进，完成后回家封口。
  - RETURN：目标 = 最近己方领地格；轨迹暴露时间超过容忍值（caution 越高越短）也触发。
  - RAID：aggression 高且邻近敌方领地边缘时，切进敌领地画一刀再回家（偷地）。
  - DEFEND：敌人头部接近自己领地/轨迹 12 格内且自己能更快截到对方轨迹 → 冲向其轨迹格。
  - FLEE：敌头与自己轨迹最近距离 < 危险阈值且在逼近 → 放弃目标直线回家。
  - 随局势升温：游戏时间每 60s 或玩家 % 每 +10，全体 aggression +0.05（上限 0.9）——对齐原版难度曲线。
  - 低概率(由 skill 反比)犯错：晚一拍 FLEE、贴边失误等，让孩子能赢。
- Bot 名字池（≥20 个原创可爱中文名，如 果冻鸭、甜圈猫、汽水熊、泡泡龙…）+ 对应 emoji 头像 + 专属 pastel 色。

## 6. 渲染（render.js）
分层：纸纹背景 → 领地层 → 轨迹 → 角色 → 粒子/飘字 → HUD。
- 背景：暖白 `#f7f2e9` + 细点阵（点阵预绘到离屏 pattern，随镜头平移重复）。地图外区域画更深的灰色+边界圆角描边，让边界可读。
- **领地**：每 owner 一个缓存 `Path2D`（脏标记时重算）：marching squares（400×400 网格上该 owner 的掩码，一致处理鞍点）→ 环简化（去共线点）→ Chaikin×2 → Path2D。每帧按相机变换 fill；先画深 15% 的同色底（y 偏移 6u）做厚度感，再画本色，边缘 2u 描边（同色系深 25%）。
- 轨迹：轨迹点列 Chaikin×1 后一次 stroke，宽 13u、round cap/join、透明度 0.5、本人色；头端画个小圆点收口。
- 角色：22u 圆角方块（本人色，深色描边+底部投影椭圆），上面居中画皮肤 emoji（22px）；**转向时车身朝航向旋转 + 过弯外倾 8°**；出领地瞬间轻微 squash&stretch。名字标签白底圆角小牌浮在头顶（玩家显示「你」+皮肤名）。#1 头顶画金皇冠（canvas 画三角冠+圆点）。
- 小地图：132px 圆角方 canvas，5Hz 重绘：把 400×400 grid 直接按 3 格/px 采样画色块，玩家位置白点脉冲，视野框细线。
- 粒子/飘字/confetti/屏震全走对象池（juice.js）。

## 7. 相机
- 跟随头部，exp 平滑 k=8；缩放 = `clamp(0.9 * (0.0066/max(share,0.0066))^0.18, 0.42, 1.0)`（share=玩家占比），每 +10% 明显拉远一档的手感，exp 平滑 k=3。死亡/击杀时 6u 震屏衰减。

## 8. HUD 与界面
- 顶部中央大百分比 `12.4%`（等宽数字，变化时弹跳）；其下小字「最高 xx.x%」。
- 右上排行榜：前 5 名（色块+名+%），#1 带 👑；玩家不在前 5 时底部追加自己行。DOM 实现，2Hz 更新。
- 左下小地图；右上角暂停键。击杀 toast：「你切断了 果冻鸭！」左上滑入淡出；被 bot 互杀也播（「汽水熊 切断了 泡泡龙」）。
- 开始界面：LOGO「圈地大冒险」，6 个皮肤大按钮（🍦🦆🍩🐱🚗👾 圆角卡片，锁定的显示解锁条件），最高纪录，大开始按钮。
- 皮肤解锁：默认 🍦；最高 10% 解锁 🦆；20% 🍩；35% 🐱；50% 🚗；100% 👾。存 storage。
- 结算界面：本局 %、击杀数、存活时间、是否新纪录（NEW RECORD 弹跳标）、鼓励文案、再来一局。
- 100% 胜利：烟花 confetti 全屏 + 「你统治了整个世界！」+ 继续游玩/再来一局。

## 9. 音效（audio.js 合成）
出领地嗖声（短噪声扫频）、占领上行琶音（Δ% 越大音阶越高）、击杀爆破、死亡下滑音、复活闪烁音、胜利大琶音、按钮点击 pop。BGM：轻快 4 音循环（大调五声音阶，木琴感三角波），音量 0.12。
